# GraphQL サーバサイド

![GQL](https://pbs.twimg.com/media/DgsXLk_X4AEKiJJ.jpg "GQL")
> https://twitter.com/NikkitaFTW/status/1011928066816462848


## use

```
yarn
yarn start

# or
npm i
npm start
```

## url

### GraphQL End point
- http://localhost:3000/graphql

### GraphQL Editor
- http://localhost:3000/graphiql

### GraphQL Voyager
- http://localhost:3000/voyager
